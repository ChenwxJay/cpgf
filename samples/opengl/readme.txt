A sample for using OpenGL in Lua and Google V8 Javascript

It's only tested on Windows with Microsoft VC.

To compile it, you need to change the related include and lib path in the VC project.

The sample requires OpenGL and GLUT.

The sample will display a colorful rotating 3D cube.
Press arrow keys (up, down, left, right) will move the cube a little bit.
All the GL/GLUT related code is written in Lua and Javascript.
